Hoy vamos a ver los bordes en css (basicos)

Primero tenemos que mostrar el borde con la siguiente linea.
--> border: 1px solid;
muestra el borde con 1 pixel de grosor y del tipo solido.

Para cambiarle el color se usa la siguiente linea.
--> border-color: black;

Los posibles estilos de borde son:
-->solid
-->dotted
-->double
-->dashed
-->groove
-->ridge
-->inset
-->outset
